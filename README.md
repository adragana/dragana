# Static Portfolio

This is a static portfolio website built with plain HTML, CSS, and JavaScript.

## Files to update

- `index.html`: edit your bio and personal links.
- `data/projects.js`: replace the sample project titles, descriptions, roles, stacks, and screenshot paths.
- `projects/*.html`: each project already has its own page and reuses shared data.
- `assets/`: replace the SVG placeholders with your real screenshots or preview images.

## How to add a new project

1. Add a new project object in `data/projects.js`.
2. Duplicate one of the files inside `projects/` and change the `data-project` value.
3. Add your images into `assets/`.

## How to preview

Open `index.html` in a browser.
